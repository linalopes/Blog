{%- comment -%} Renders category badge list for a post/page. Usage: {% include catlist.html categories=post.categories %} Params: categories – array of category names (required) link_class – CSS classes for the tag (default: "text-muted smoothscroll") downcase – if "true", display text is downcased (default: false) {%- endcomment -%} {%- for category in include.categories -%} {% if include.downcase == "true" %}{{ category | downcase }}{% else %}{{ category }}{% endif %}, {%- endfor -%}
